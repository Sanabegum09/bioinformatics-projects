# Molecular Docking Projects

This directory contains projects focused on molecular docking, virtual screening, and ligand-receptor binding studies for drug discovery.

## Project Categories

### 1. Protein Preparation
- Receptor structure optimization
- Hydrogen addition and charge assignment
- Grid generation for docking

### 2. Ligand Preparation
- Ligand database creation
- 3D structure generation
- Conformer generation

### 3. Molecular Docking Studies
- Molecular docking simulations
- Binding affinity predictions
- Docking score analysis and ranking

### 4. Virtual Screening
- Large-scale ligand screening
- Hit identification and ranking
- Lead compound selection

### 5. MMGBSA & MMPBSA Analysis
- Binding free energy calculations
- Energy decomposition analysis
- Per-residue contribution analysis

## Tools Used
- AutoDock Vina
- AutoDock 4
- GROMACS with docking modules
- MMGBSA/MMPBSA utilities

## Getting Started
See individual project folders for specific docking protocols and analysis scripts.

## Key Metrics
- Binding Free Energy (ΔG)
- Docking Score / Affinity
- RMSD (Root Mean Square Deviation)
- Hydrogen bonds count
- Hydrophobic interactions
