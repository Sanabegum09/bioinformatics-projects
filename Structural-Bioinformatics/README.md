# Structural Bioinformatics Projects

This directory contains projects focused on protein structure analysis, validation, and structure-based drug design.

## Project Categories

### 1. Protein Structure Prediction
- AlphaFold2 predictions
- SWISS-MODEL homology modeling
- Structure quality assessment

### 2. Structure Validation
- Ramachandran plot analysis
- Z-score calculations
- Quality metrics (RMSF, B-factors)

### 3. Active Site Analysis
- Binding site identification
- Pocket characterization
- Ligand interaction analysis
