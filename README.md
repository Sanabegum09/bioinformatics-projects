## 🔬 Project 1 — Molecular Docking & Virtual Screening

> **Core computational engine behind all published drug discovery work**

[![Folder](https://img.shields.io/badge/📁_View_Folder-Molecular--Docking-1A5276?style=for-the-badge)](./Molecular-Docking)
[![Script](https://img.shields.io/badge/🐍_Main_Script-autodock__vina__pipeline.py-117A65?style=for-the-badge)](./Molecular-Docking/autodock_vina_pipeline.py)
[![Study](https://img.shields.io/badge/📖_Published_Study-SARS--CoV--2_Mpro-C0392B?style=for-the-badge)](./Molecular-Docking/SARS-CoV2-Mpro-Study)

### What it does
**`autodock_vina_pipeline.py`** — Fully automated high-throughput virtual screening pipeline using AutoDock Vina. Input a prepared receptor and a folder of ligand PDBQT files; the script runs all dockings, parses binding affinities, ranks compounds, flags hits below a user-defined cutoff, and exports a clean results CSV with docked poses.

### Applied in research
Used to screen natural phytochemical libraries against:
- **DNMT1/3A/3B** (Head & Neck Cancer — *Results in Chemistry* 2025)
- **SARS-CoV-2 Mpro** (COVID-19 — *ResearchGate* 2024)
- **ALK** (NSCLC — *Under Review*)
- **BMI1** (Bladder Cancer — *In Preparation*)
- 6+ additional disease targets

```bash
# Screen a compound library against a target receptor
python Molecular-Docking/autodock_vina_pipeline.py \
    --receptor  receptor.pdbqt           \
    --ligands_dir  phytochemical_library/ \
    --center_x -26.5 --center_y 15.2 --center_z -16.8 \
    --size_x 20 --size_y 20 --size_z 20  \
    --exhaustiveness 8  --top_n 20        \
    --output  screening_results/
```

<details>
<summary><b>📋 Output Files</b></summary>

| File | Description |
|------|-------------|
| `screening_results_all.csv` | Full ranked compound list with all binding modes |
| `screening_results_top20.csv` | Top 20 hits sorted by binding affinity (kcal/mol) |
| `docked_poses/*.pdbqt` | Best docked pose for each compound |
| `vina_logs/*.log` | Raw AutoDock Vina output for each run |
| `pipeline_YYYYMMDD.log` | Full run log with per-compound status |

</details>

## 💊 Project 2 — ADMET Prediction: pkCSM + SwissADME

> **Replicates the exact ADMET workflow from** *Results in Chemistry* **(2025)**

[![Folder](https://img.shields.io/badge/📁_View_Folder-ADMET--Prediction-8E44AD?style=for-the-badge)](./ADMET-Prediction)
[![Script](https://img.shields.io/badge/🐍_Main_Script-admet__screening.py-8E44AD?style=for-the-badge)](./ADMET-Prediction/admet_screening.py)
[![pkCSM](https://img.shields.io/badge/🌐_Server-pkCSM-1A5276?style=for-the-badge)](https://biosig.lab.uq.edu.au/pkcsm/)
[![SwissADME](https://img.shields.io/badge/🌐_Server-SwissADME-117A65?style=for-the-badge)](https://www.swissadme.ch/)

### Servers used
| Server | URL | Predicts |
|--------|-----|---------|
| **pkCSM** | https://biosig.lab.uq.edu.au/pkcsm/ | Absorption · Distribution · Metabolism · Excretion · Toxicity |
| **SwissADME** | https://www.swissadme.ch/ | Physicochemical · Drug-likeness · Bioavailability radar |

### Step-by-step workflow

```
① Run virtual screening  →  get top docking hits (SDF / SMILES)
         ↓
② python admet_screening.py --prepare --smiles hits.sdf
         ↓  generates compounds_for_servers.smi
③ Submit to pkCSM (Batch tab)  →  download pkcsm_results.csv
         ↓
④ Submit to SwissADME          →  download swissadme_results.csv
         ↓
⑤ python admet_screening.py \
       --pkcsm pkcsm_results.csv \
       --swissadme swissadme_results.csv \
       --plot
         ↓
⑥ Get: admet_heatmap.png · bioavailability_radar.png · toxicity_summary.png
        admet_drug_like_hits.csv  ← publication-ready output
```

<details>
<summary><b>📋 Properties Predicted</b></summary>

| Category | pkCSM | SwissADME |
|----------|-------|-----------|
| Absorption | Caco-2, HIA (%), P-gp substrate/inhibitor | GI absorption (High/Low) |
| Distribution | VDss, BBB permeability, CNS permeability | BBB permeant (Yes/No) |
| Metabolism | CYP1A2/2C9/2C19/2D6/3A4 inhibition & substrate | CYP inhibition panel |
| Excretion | Renal OCT2, Total clearance | — |
| Toxicity | AMES, hERG I/II, Hepatotoxicity, LD50, Skin sensitisation | PAINS, Brenk alerts |
| Drug-likeness | — | Lipinski Ro5, Veber, Egan, Ghose, QED, Bioavailability score |

</details>
