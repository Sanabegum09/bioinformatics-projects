## Projects

### 🔬 Molecular Docking & Virtual Screening

> Automated high-throughput docking pipeline — the core computational engine behind all published drug discovery work.

**[`autodock_vina_pipeline.py`](./Molecular-Docking/autodock_vina_pipeline.py)** — Batch AutoDock Vina virtual screening. Takes a receptor and a library of compounds, runs all dockings in parallel, parses binding affinities, ranks hits, and exports a clean CSV. Handles PDBQT preparation, Vina configuration, output parsing, and logging automatically.

**[`SARS-CoV2-Mpro-Study/`](./Molecular-Docking/SARS-CoV2-Mpro-Study/)** — Full documented workflow from the 2024 publication: receptor preparation from PDB 6LU7, phytochemical library screening, MD simulation commands (GROMACS), and MM-PBSA analysis.

```bash
python Molecular-Docking/autodock_vina_pipeline.py \
    --receptor  receptor_prepared.pdbqt     \
    --ligands_dir  library_pdbqt/           \
    --center_x -26.5 --center_y 15.2 --center_z -16.8 \
    --exhaustiveness 8  --top_n 20          \
    --output  screening_results/
```

### 💊 ADMET Prediction: pkCSM + SwissADME

> Reflects the exact ADMET workflow used in *Results in Chemistry* (2025) — pkCSM for ADME/Tox, SwissADME for drug-likeness and bioavailability radar.

**[`admet_screening.py`](./ADMET-Prediction/admet_screening.py)** — Prepares SMILES input for both servers, then parses and integrates the downloaded CSV results. Outputs an ADMET heatmap, bioavailability radar, toxicity summary chart, and a filtered drug-like hits table.

```
Step 1 │ Prepare SMILES  →  python admet_screening.py --prepare --smiles hits.sdf
Step 2 │ Submit to pkCSM    →  https://biosig.lab.uq.edu.au/pkcsm/   → download CSV
Step 3 │ Submit to SwissADME →  https://www.swissadme.ch/            → download CSV
Step 4 │ Integrate & plot  →  python admet_screening.py --pkcsm pkcsm.csv \
       │                                                 --swissadme swiss.csv --plot
```

| Predicted Property | Source |
|--------------------|--------|
| Caco-2 · HIA · P-gp · VDss · BBB · CNS permeability | pkCSM |
| CYP1A2 / 2C9 / 2C19 / 2D6 / 3A4 inhibition & substrate | pkCSM + SwissADME |
| AMES mutagenicity · hERG I/II · Hepatotoxicity · Skin sensitisation · LD50 | pkCSM |
| Lipinski Ro5 · Veber · Egan · Ghose · QED · PAINS · Bioavailability radar | SwissADME |

### 🧬 NGS Analysis: Alzheimer's Disease

> Three complete NGS pipelines applied to publicly available Alzheimer's disease datasets, demonstrating RNA-seq, miRNA-seq, and WES/DNA-seq analysis from raw data to biological interpretation.

| Pipeline | Script | Dataset | Key Output |
|----------|--------|---------|------------|
| **RNA-seq** | [`alzheimers_rnaseq_pipeline.R`](./NGS-Analysis/Alzheimers-RNA-seq/) | GSE110226 (NCBI GEO) | DESeq2 DE genes · KEGG/GO enrichment · AD hallmark gene heatmap |
| **miRNA-seq** | [`alzheimers_mirna_pipeline.R`](./NGS-Analysis/Alzheimers-miRNA-seq/) | GSE46579 (NCBI GEO) | DE miRNAs · miR-29/132/146a spotlight · target gene export |
| **DNA-seq / WES** | [`alzheimers_dnaseq_pipeline.sh`](./NGS-Analysis/Alzheimers-DNA-seq/) | ADNI phs000572 | GATK4 variants · APOE ε4 genotyping · ANNOVAR annotation |

```r
# RNA-seq: auto-downloads GSE110226 from NCBI GEO, runs full pipeline
Rscript NGS-Analysis/Alzheimers-RNA-seq/alzheimers_rnaseq_pipeline.R \
    --output results_rnaseq/
```

```bash
# DNA-seq: full GATK4 best-practices WES pipeline
bash NGS-Analysis/Alzheimers-DNA-seq/alzheimers_dnaseq_pipeline.sh \
    SAMPLE001  reads_R1.fastq.gz  reads_R2.fastq.gz
```

## Quick Start

```bash
# 1. Clone
git clone https://github.com/Sanabegum09/bioinformatics-projects.git
cd bioinformatics-projects

# 2. Python (Molecular Docking + ADMET)
pip install -r Molecular-Docking/requirements.txt
```
